## Symptom
Build fails with `sh: 1: vite: not found`.

## Patch
```patch
TARGET FILE: package.json

FIND:
"build": "vite build"

REPLACE WITH:
"build": "vite build"

Notes

Ensure vite exists in dependencies, not only devDependencies.
```
