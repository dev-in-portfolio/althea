## Overview
Causality is a tool for mapping and visualizing causal relationships between operational events and incidents.

## Features
- **Graph Utilities**: Advanced utilities for managing node relationships.
- **Incident Mapping**: Visualize how one event leads to another.
- **Impact Analysis**: Identify potential ripple effects in operational flows.

## How to Use
1. Add operational events as nodes.
2. Define causal links between nodes.
3. Run impact analysis to see cascading dependencies.
