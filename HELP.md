## Overview
NeonScope is a Python-powered database exploration tool for querying and visualizing complex datasets.

## Features
- **Query Engine**: Run custom SQL or predefined queries against the database.
- **Result Visualization**: View query outputs in organized, interactive tables.
- **Metadata Explorer**: Inspect database schema and table structures.

## How to Use
1. Select a predefined query from the menu or input your own.
2. Execute the query to see the live data output.
3. Export results to CSV or JSON for external analysis.
