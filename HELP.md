## Overview
Pocket Pass is a Next.js application for generating and viewing digital IDs and operational passes.

## Features
- **Pass Viewer**: Specialized interface for clear, high-contrast ID display.
- **Dynamic Generation**: Create passes based on live operator data.
- **Modern Stack**: Built with Next.js 14 for optimal performance.

## How to Use
1. Input the data for the required ID or pass.
2. Preview the pass in the specialized viewer.
3. Generate a digital copy or print-ready version for physical use.
