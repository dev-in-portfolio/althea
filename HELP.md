## Overview
Switchboard is a central directory and search hub for operational targets and service endpoints.

## Features
- **Target Directory**: Comprehensive list of operational targets.
- **Fast Search**: Built-in search utility for quick target identification.
- **Kind Categorization**: Group targets by service type or priority.

## How to Use
1. Use the Search bar to find a specific target or service.
2. Filter by 'Kind' to see all targets in a specific category.
3. Review target metadata for connectivity and status details.
