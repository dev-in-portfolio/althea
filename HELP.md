## Overview
Chronicle provides a chronological operational log and timeline viewer for Althea incidents and activities.

## Features
- **Timeline Visualization**: View events in sequence.
- **Status Filtering**: Filter by incident or event type.
- **Persistence**: Server-backed storage for historical data.

## How to Use
1. Launch the dashboard to see current operational logs.
2. Filter events using the top navigation bar.
3. Add new entries via the event creator modal.
