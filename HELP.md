## Overview
Qwik Atlas is a high-speed explorer for navigating Althea's archive nodes, signal relays, and infrastructure bridges.

## Features
- **Resumable UI**: Powered by Qwik for instant interactivity and low bundle size.
- **Live Filtering**: Filter infrastructure nodes by status (active, paused, retired).
- **Device Persistence**: Automatic generation and storage of device-specific access keys.

## How to Use
1. Search the atlas for specific archive nodes or signal relays.
2. Use the status filters to identify infrastructure that requires attention.
3. Toggle between grid and list layouts for optimal data viewing.
