## Overview
SessionMint is a Next.js application designed to generate and "mint" secure operational sessions for Althea users.

## Features
- **Session Minting**: Generate unique, time-limited operational sessions.
- **Fast Rendering**: Built with Next.js for high-speed session distribution.
- **Secure Persistence**: Integrated session tracking to prevent unauthorized reuse.

## How to Use
1. Authenticate and request a new session from the minting dashboard.
2. Use the minted session token to access protected Althea resources.
3. View your session history and active tokens in the user profile.
