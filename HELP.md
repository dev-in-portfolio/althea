## Overview
Exhibit-Forge is a Django-based management system for creating and editing exhibits for the Digital Science Museum.

## Features
- **Exhibit Modeling**: Robust data models for scientific exhibits.
- **Admin Interface**: Full-featured backend for exhibit curators.
- **Relationship Mapping**: Link exhibits to specific museum halls.

## How to Use
1. Access the admin dashboard to create new exhibits.
2. Populate exhibit metadata, including sources and descriptions.
3. Map exhibits to their respective halls for site-wide updates.
