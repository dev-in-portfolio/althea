## Overview
Codex of H is a specialized linguistic atlas and dictionary for aed-based glyphs and translations.

## Features
- **Glyph Dictionary**: Comprehensive database of glyph meanings.
- **Translation Engine**: Maps aed word translations to modern equivalents.
- **Linguistic Data**: Pre-loaded with spellings and word-level data.

## How to Use
1. Search for a specific glyph or word in the search interface.
2. View translations, spellings, and dictionary entries.
3. Use the glyph atlas for visual identification of script elements.
