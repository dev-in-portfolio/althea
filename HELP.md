## Overview
Snapshot Vault is a NestJS API for creating, storing, and managing versioned snapshots of operational documents.

## Features
- **Document Versioning**: Track changes and maintain historical snapshots of critical documents.
- **AuthLite Guarding**: Secure API endpoints with device-key authentication.
- **Scalable Backend**: Built with NestJS for robust document management.

## How to Use
1. Submit a document to the snapshots endpoint to create a new version.
2. Retrieve historical snapshots via the versioning index.
3. Use the document controller to manage active and archived assets.
