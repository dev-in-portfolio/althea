## Overview
Patch-Book is an Eleventy-powered static site for tracking operational patches, updates, and associated risks.

## Features
- **Patch Indexing**: Chronological log of system and operational patches.
- **Risk Assessment**: Associate specific risks with each patch entry.
- **Static Site**: Fast, searchable, and easy to deploy as a documentation hub.

## How to Use
1. Navigate the Patch Index to see recent updates.
2. Click into individual patches for full risk assessments.
3. Use the search bar to find patches by tag or version.
