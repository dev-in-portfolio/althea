## Overview
Rule Furnace is a Deno-powered environment for testing and evaluating complex operational rules and logic.

## Features
- **Rule Evaluation**: Integrated engine for running live tests on logic rules.
- **Test Harness**: Specialized UI (islands) for simulating rule conditions.
- **Fresh Reactivity**: Instant feedback on rule changes and evaluation runs.

## How to Use
1. Define a new rule in the Rule Form.
2. Use the Test Harness to input variables and run evaluation simulations.
3. Review the execution logs in the 'Runs' section to verify rule accuracy.
