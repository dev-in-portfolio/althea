## Overview
Contradict is a Next.js application for identifying and managing conflicting statements and logical paradoxes.

## Features
- **Statement Management**: Input and tag various statements.
- **Conflict Detection**: Engine-driven analysis to find contradictions.
- **Dictionary Integration**: Built-in dictionary for semantic analysis.

## How to Use
1. Navigate to the Statements page to add new entries.
2. Use the Conflicts page to run the detection engine.
3. Review highlighted contradictions in the analysis view.
