## Overview
TapForge is a Flutter-based application for 'tapping' into Althea data streams and monitoring real-time throughput.

## Features
- **Data Tapping**: Connect to and monitor live operational data streams.
- **Node Backend**: Secure server integration for managing multiple data taps.
- **Visual Throughput**: Real-time visualization of data flow and volume.

## How to Use
1. Initialize a new 'tap' by specifying the data stream target.
2. Monitor the live flow on the mobile or web interface.
3. Log throughput anomalies for further investigation by the Althea engineering team.
