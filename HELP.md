## Overview
This branch serves as a deployment target for Streamlit-based data applications within the Althea ecosystem.

## Features
- **Python Dashboarding**: Ready to host interactive data apps built with Streamlit.
- **Fast Prototyping**: Ideal for quick visualization of Althea operational metrics.
- **Static Fallback**: Includes a standard landing page for when the Python app is not active.

## How to Use
1. Deploy your Streamlit app using the provided Althea Python templates.
2. Access interactive graphs and metrics via the web interface.
3. Use the sidebar controls to filter and drill down into operational data.
