## Overview
Hallmap-Studio is a Python application for designing and auditing virtual museum hall layouts.

## Features
- **Hall Audits**: Built-in tools for auditing hall data integrity.
- **Import/Export**: Move hall maps between formats (JSON, CSV).
- **Studio Interface**: Visual workspace for hall organization.

## How to Use
1. Import an existing hall map or start a new studio project.
2. Use the audit tool to check for missing exhibit links.
3. Export the finalized hall configuration for museum deployment.
