## Overview
Angle is a specialized text classification and validation utility designed for Althea operations.

## Features
- **Text Classification**: Automated categorization of operational text.
- **Validation**: Middleware for securing and validating user input keys.
- **Database Sync**: Built-in SQLite/MySQL sync for classification results.

## How to Use
1. Authenticate using your user key.
2. Submit text snippets for classification.
3. Review the categorized output via the classification utility.
