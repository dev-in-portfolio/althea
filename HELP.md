## Overview
DiffLens is a FastAPI-powered utility for high-precision comparison of text and JSON datasets.

## Features
- **JSON Diff**: Specialized engine for comparing complex nested objects.
- **Text Comparison**: Standard line-by-line and character-level diffing.
- **API First**: Exposes robust endpoints for programmatic comparisons.

## How to Use
1. Input the 'Source' and 'Target' content.
2. Select the diff mode (JSON or Text).
3. View the highlighted differences in the side-by-side viewer.
