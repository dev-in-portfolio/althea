## Overview
TimeSlice is a SvelteKit application for slicing and analyzing operational data based on specific time intervals.

## Features
- **Time Slicing**: View operational data across custom-defined time segments.
- **Historical Comparison**: Compare data slices from different time periods.
- **Reactive Visuals**: Built with Svelte for smooth exploration of time-series data.

## How to Use
1. Define the time interval you wish to slice (e.g., hourly, daily).
2. Browse the resulting data slices to identify patterns or anomalies.
3. Export specific time slices for use in historical trend analysis.
