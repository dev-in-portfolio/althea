## Overview
SchemaGate is a FastAPI-powered gateway for validating and normalizing data against Althea schema standards.

## Features
- **Schema Validator**: Strict validation of incoming data batches.
- **Normalizer**: Automatically converts non-standard data into canonical formats.
- **Schema Store**: Central repository for all active operational schemas.

## How to Use
1. Submit your data batch to the validation endpoint.
2. Review any validation errors or normalization warnings.
3. Use the CLI tool to manage and update schemas in the central store.
