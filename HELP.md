## Overview
SignalBoard is a Vue-based dashboard for visualizing and managing real-time operational signals.

## Features
- **Vue 3 Interface**: Reactive and modular signal display components.
- **Live Updates**: High-speed signal monitoring via dedicated API integration.
- **Device Key Guard**: Secure dashboard access using localized device keys.

## How to Use
1. Open the board to see a live view of all active operational signals.
2. Filter signals by type or priority to focus on critical data.
3. Use the router to navigate between different signal groups and historical logs.
