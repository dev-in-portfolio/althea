## Overview
LineFlow is a high-concurrency timer and preset management system for operational line tracking.

## Features
- **Active Timers**: Supports up to 50 concurrent active timers per session.
- **Presets**: Store up to 200 custom timer presets for quick access.
- **Postgres Backend**: Reliable persistence for timer logs and history.

## How to Use
1. Select a preset or create a custom timer.
2. Start the timer to track an operational process.
3. Monitor multiple active timers simultaneously from the dashboard.
