## Overview
Dockyard is a project organization tool designed with an Electron-style interface for managing operational 'containers' and workspaces.

## Features
- **Workspace Management**: Organize tasks and resources into distinct dockyards.
- **Preload Utilities**: Fast, desktop-like responsiveness for web environments.
- **Resource Tracking**: Monitor the 'load' and status of your projects.

## How to Use
1. Initialize a new Dockyard for your current project.
2. Add resources, task lists, and metadata.
3. Switch between dockyards using the sidebar navigator.
