## Overview
LatchList is a Next.js application designed for task persistence and "latching" operational goals.

## Features
- **Goal Latching**: Persistent task tracking with status monitoring.
- **React-Powered UI**: Responsive and interactive goal management.
- **Next.js 14**: Utilizing the latest App Router for optimal routing.

## How to Use
1. Add a new operational goal or task to the list.
2. "Latch" the task to indicate it is active and being monitored.
3. Mark tasks as complete to archive them in the history log.
