## Overview
ReleaseGallery is a release vault used to store artifact manifests, checksums, and version history.

## Features
- **Artifact Tracking**: Detailed manifests for every software release.
- **Security Focus**: Provides checksums for artifact verification.
- **Project Taxonomy**: Organize releases by project name and version.

## How to Use
1. Browse releases by project name.
2. Select a specific version to view its artifact manifest.
3. Use the provided checksums to verify your downloaded binaries.
