## Overview
RouteForge Console is a Python utility for normalizing, scoring, and performing QA on operational routes.

## Features
- **Route Normalization**: Standardizes diverse route data formats.
- **Scoring Engine**: Evaluates route efficiency and reliability.
- **QA Module**: Built-in tools for auditing route integrity and completeness.

## How to Use
1. Load your route data into the console.
2. Run the normalizer to ensure data consistency.
3. Review the quality score and QA flags to identify routes that need recalibration.
