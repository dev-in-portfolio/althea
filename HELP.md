## Overview
Compression is a text summarization tool designed to condense large bodies of text into multi-level summaries.

## Features
- **Multi-Level Summarization**: Choose the intensity of compression.
- **Sentence Control**: Set the maximum number of sentences for the output.
- **User Persistence**: Stores your session key locally for quick access.

## How to Use
1. Paste your text into the text area.
2. Select the compression level and maximum sentences.
3. Submit to generate a condensed version of the content.
