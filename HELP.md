## Overview
Momentum is a SvelteKit-based progress tracker for monitoring long-term operational trends.

## Features
- **SvelteKit Speed**: Fast, reactive progress updates.
- **Server-Side Logic**: Secure data handling via +page.server.ts.
- **Trend Analysis**: Visualize momentum and progress over time.

## How to Use
1. Input your current operational metrics.
2. View the momentum graph to see trends and progress.
3. Update metrics regularly to maintain an accurate trendline.
