## Overview
Pocket Dossier is a Flutter-based personal record manager for mobile and web.

## Features
- **Cross-Platform**: Consistent experience across mobile and web.
- **Record Management**: Create and organize detailed personal dossiers.
- **Node Backend**: Secure server integration for data synchronization.

## How to Use
1. Create a new dossier for an individual or project.
2. Add records, tags, and metadata to the dossier timeline.
3. Sync data with the server to access records across all your devices.
