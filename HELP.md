## Overview
Paradox Vault is a curation repository for notes, bookmarks, and operational reports.

## Features
- **Serverless Integration**: Powered by Netlify Functions for dynamic data management.
- **Report Curation**: Archive and view detailed curation reports.
- **Bookmark System**: Save and organize essential operational resources.

## How to Use
1. Access the vault to browse saved notes and bookmarks.
2. Submit new reports via the curation interface.
3. Use the search index to find specific archived materials.
