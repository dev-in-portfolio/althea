## Overview
Blueprint Library acts as a central repository for operational templates and standard operating procedures (SOPs).

## Features
- **Template Versioning**: Track changes to operational blueprints.
- **Search & Filter**: Find specific templates by tags or keyword.
- **Django Backend**: Secure, scalable management of textual assets.

## How to Use
1. Search for a specific SOP or blueprint in the search bar.
2. Download or view the latest version of the template.
3. Curators can add new blueprints via the admin panel.
