## Overview
RelayRoom is a NestJS-powered API for managing operational 'rooms' and collaborative notes.

## Features
- **Room Management**: Create and organize rooms for different projects or teams.
- **Note Synchronization**: Real-time endpoints for collaborative note-taking.
- **Secure Access**: Integrated device-key guards and request-id middleware.

## How to Use
1. Create a new RelayRoom for your project via the API.
2. Add operational notes and updates to the room's stream.
3. Use the search endpoints to find historical notes across different rooms.
