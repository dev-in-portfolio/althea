## Overview
Room-Key is a Next.js application for managing access keys and sessions for operational rooms.

## Features
- **Key Management**: Generate and track unique access keys for various project rooms.
- **Session Control**: Monitor and expire active room sessions.
- **Next.js 14**: Modern, responsive UI for fast key distribution.

## How to Use
1. Request a key for a specific room via the dashboard.
2. Use the generated key to authenticate into the corresponding operational space.
3. Admins can monitor active keys and revoke access as needed.
