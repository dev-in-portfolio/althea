## Overview
Switchboard is a Vue 3 application that acts as a central hub for toggling and routing operational signals.

## Features
- **Signal Routing**: Dynamically route signals to different backend targets.
- **Toggle Controls**: Simple, visual interface for activating or deactivating operational nodes.
- **Vue 3 Speed**: Fast, reactive dashboard for high-pressure operational environments.

## How to Use
1. Monitor the main dashboard for an overview of all active routes.
2. Use the toggles to instantly route signals to their required destinations.
3. Review the connection logs to ensure all signals are being routed correctly.
