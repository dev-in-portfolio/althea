## Overview
QueJudge is a FastAPI-powered application for scoring, explaining, and judging operational queries and queues.

## Features
- **Query Scoring**: Algorithmic scoring of query efficiency and accuracy.
- **Explain Engine**: Provides detailed explanations for query judgments.
- **API First**: High-performance FastAPI endpoints for integration.

## How to Use
1. Submit a query or queue to the scoring endpoint.
2. Review the judgment and efficiency score.
3. Use the explain feature to understand the reasoning behind the score.
