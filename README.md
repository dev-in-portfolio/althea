# Althea

Qwik Atlas is a Qwik City collection browser with URL-driven filters and Neon-backed saved views.

Quick start:

1. Apply `sql/001_qwik_atlas.sql` to your Neon database.
2. Create a `.env` from `.env.example` and set `DATABASE_URL`.
3. Install deps: `npm install`
4. Run API server: `npm run server`
5. Run client: `npm run dev`
