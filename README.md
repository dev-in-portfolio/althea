# Althea

Signal Board is a Vue + Neon dashboard for live status tiles with rule-driven alerts.

Quick start:

1. Apply `sql/001_signalboard.sql` to your Neon database.
2. Create a `.env` from `.env.example` and set `DATABASE_URL`.
3. Install deps: `npm install`
4. Run API server: `npm run server`
5. Run client: `npm run dev`
